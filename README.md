# tampermonkey-ozon-tickets-export

Tampermonkey userscript for making ticket export and operational review faster inside an existing browser-based workflow.

## Why this project exists

Some operational systems are usable enough for manual work, but painful when the same filtering, copying, exporting, and SLA-checking steps have to be repeated every day.

This repository is built around that kind of problem: improve the operator workflow without replacing the underlying system.

## What it is

`tampermonkey-ozon-tickets-export` is a browser-side helper script.

It is best framed as a focused operator tool that augments an existing ticket interface and helps move ticket information into a more usable form for review, reporting, or follow-up.

## Best use cases

- speeding up repeated ticket export work;
- making ticket review more visible;
- reducing manual copy-paste from a browser interface;
- supporting lightweight SLA or operational control workflows.

## Technical positioning

This is not a backend service and not a replacement for the source system.

It is a practical browser automation layer: close to the page, close to the operator, and intentionally narrow in scope.

## Portfolio value

This repo strengthens the portfolio because it shows applied automation in a real-world shape:

- legacy or closed interfaces;
- repeated manual work;
- browser-side augmentation;
- practical value without needing a large application.
